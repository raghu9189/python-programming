## 🛠️ Getting Started

### 1. Set Up a Virtual Environment (Recommended)

Virtual environments isolate your project dependencies so they don't conflict with global Python installations.

```bash
# Create a virtual environment named 'venv'
python3 -m venv venv

# Activate the virtual environment
# On macOS/Linux:
source venv/bin/activate

# On Windows (Command Prompt):
# venv\Scripts\activate.bat

# On Windows (PowerShell):
# venv\Scripts\Activate.ps1
```

### 2. Common Data Science Libraries to Install

You can install the essential toolkit of libraries for data exploration, analysis, visualization, and modeling as you need them:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```
